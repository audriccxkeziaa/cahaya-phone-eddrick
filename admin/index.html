Tambahkan ke Home Screen iPhone/iPad kamu

  1. 1
    Tap tombol Share di bagian bawah Safari.
  2. 2
    Scroll, lalu pilih "Add to Home Screen"
  3. 3
    Tap "Add" di pojok kanan atas.
  1. 1
    Klik tombol Share di toolbar Safari (atau menu File).
  2. 2
    Pilih "Add to Dock".
  3. 3
    Klik "Add" untuk konfirmasi — app muncul di Dock.
  1. 1
    Cek icon install di kanan address bar — klik, lalu pilih Install.
  2. 2
    Atau buka menu ⋮ di pojok kanan atas → Cast, save, and share → Install page as app.
  3. 3
    Kalau icon Install gak muncul, kemungkinan app udah ke-install sebelumnya. Buka chrome://apps untuk cek.
